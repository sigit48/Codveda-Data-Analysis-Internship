## 🛠️ Level 1: Foundation (Current Progress)

### Task 1: Data Cleaning & Preprocessing 🧹
In this task, I handled raw datasets with significant inconsistencies to ensure they are "Analysis-Ready".

**Key Implementations:**
* **Automated Data Profiling:** Systematically identified missing values, duplicates, and data type mismatches.
* **Sanity Checks:** Performed logical consistency verification (e.g., ensuring no negative values in call durations or charges).
* **Standardization:** Converted all feature names to `snake_case` and normalized categorical strings for better query performance.
* **Datasets Processed:**
    * `Customer Churn`: Binary encoding for categorical features and feature engineering for "Total Charges".
    * `Sentiment Analysis`: Text normalization and handling of social media metadata.

**Tools Used:** `Python`, `Pandas`, `NumPy`.
